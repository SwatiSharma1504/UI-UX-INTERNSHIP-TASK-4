Design Principles
05 — Principles

6 UX Principles for Spatial Design

01
ZERO OCCLUSION

UI elements never block important real-world or virtual content. Peripheral placement keeps the center of vision always clear for primary tasks.

02
DEPTH CLARITY

Every UI layer exists at a specific depth plane. Near UI (0.5m), mid UI (1–2m), far UI (3m+) — preventing vergence-accommodation conflict.

03
MULTIMODAL INPUT

No single input modality is required. Hand, gaze, voice, controller — any combination works for any action, accommodating all accessibility needs.

04
FORGIVENESS FIRST

All actions are reversible. Accidental triggers have a 300ms grace window. Destructive actions require a two-gesture confirmation sequence.

05
PHYSICS GROUNDING

Virtual objects obey physical metaphors — weight, inertia, bounce. Familiar physical behaviour reduces learning curve to near zero.

06
COMFORT BUDGET

No interaction requires head movement beyond 30° from centre. Arm-reach UI respects the ergonomic "golden zone" for sustained comfortable use.
